+ Minimal KernelSU version: 10940
+ Minimal KernelSU Manager (ksud) version: 11575
+ It is recommended to use the metamodule
  
### Magisk

+ Minimal version: 26402

### APatch

+ Minimal version: 10568

## Compatibility
Only OnePlus Ace 5 device (PKG110)
