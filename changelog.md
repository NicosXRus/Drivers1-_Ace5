- **Added the option to select the mounting method when installing the module**

## V2.1.4 (Updated 22 May, 2026)
- Sources Drivers updated from ColorOS 16.0.5.701
- **Added the option to select the mounting method when installing the module** 
> **Legacy mode** — Old mounting via manual bind. Doesn't work in OverlayFS.
> **New mode** — Mounting via automatic scanning and mounting via bind. Works with OverlayFS and metamodules.
> **Mountify mode** — designed for mounting in Mountify
- No longer in use power_profile.xml from ColorOS. Now the original one is modified and installed like this
- During subsequent module updates, the installation script will automatically select the required mounting method
- Fixed switching modes using volume keys
